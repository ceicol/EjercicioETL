# Migración de datos al modelo RIC
Ejercicio practico que ilustra como realizar la migración de datos al modelo de LADM-COL RIC.



- [Lineamiento Reporte de información catastral](https://igac.gov.co/es/catastro-multiproposito/reporte-de-informacion-catastral) 
- [Modelo RIC](https://gitlab-ladm-col.igac.gov.co/root/LADM_COL/-/tree/lev_cat_v2/Catastro_Multiproposito/2_Aplicacion/5_RIC_dev)
- [Requisitos de software](https://bit.ly/prerrequisitos_ladm)
  - [PostgreSQL + PostGIS](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)
  - [QGIS 3.22](https://qgis.org/es/site/forusers/download.html)
  - [Plugin QGIS Model Baker](https://github.com/SwissTierrasColombia/QgisModelBaker/releases/download/v6.1.1.5/QgisModelBaker_6115.zip)
  - [Dbeaver](https://dbeaver.io/)
